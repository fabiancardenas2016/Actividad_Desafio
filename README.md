# 🎮 Juego de Adivinanza en JavaScript
Aplicación web desarrollada con HTML, CSS y JavaScript en la que el usuario debe adivinar un número secreto generado aleatoriamente dentro de un rango definido. El juego cuenta con un sistema de 3 intentos máximos, proporcionando pistas cuando el número ingresado es mayor o menor que el valor correcto. La interfaz incluye un campo de entrada numérica, opciones para reintentar y reiniciar la partida, además de una experiencia visual simple e intuitiva.

## Características principales:
•	🔢 Ingreso de números mediante un campo numérico y Generación aleatoria de un número secreto.
•	🎨 Interfaz moderna con fuentes personalizadas de Google Fonts. 
•	🎯 Validación de números dentro del rango permitido.
•	💡 Pistas para orientar al usuario (número más alto o más bajo).
•	❤️ Sistema de intentos limitados.
•	🏆 Mensajes de victoria y derrota.
•	🔄 Reinicio automático para comenzar una nueva partida.
•	📱 Interfaz simple, intuitiva y adaptable a distintos dispositivos.

## Tecnologías utilizadas:
•	HTML5
•	CSS3
•	JavaScript 

# Resumen 
🎮 Juego interactivo de adivinanza desarrollado con HTML, CSS y JavaScript. El usuario debe descubrir un número secreto mediante intentos sucesivos, con opciones para reiniciar la partida y una interfaz sencilla y responsiva.
